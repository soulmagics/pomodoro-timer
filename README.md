# 🍎 Apple Style Pomodoro Stickman Timer

An elegant, single-file Pomodoro timer inspired by Apple's minimalist design language. Featuring a charming, highly animated stickman that visually guides you through your deep work sessions and well-deserved breaks.

![Stickman Animation Preview](https://github.com/user-attachments/assets/your-preview-here) <!-- Replace with actual image path if hosted -->

## ✨ Key Features

- **Apple Minimalism**: A clean, premium UI with glassmorphism, smooth gradients, and refined typography.
- **Lively Stickman Animation**:
    - **0-15s (Chilling)**: The stickman idles with random witty behaviors like dancing, stretching, or foot-tapping.
    - **15-50s (Orbiting)**: To show progress, the stickman actively walks in a circular orbit around the current minute marker.
    - **50-60s (Sprinting)**: As the minute concludes, it sprints along the remaining orbit.
    - **60s (The Leap)**: At exactly one minute, it performs an elastic 'boing' jump across to the next marker.
- **Circular Progress Markers**: 25 dots for Work (Focus) and 5 dots for Break (Rest), providing intuitive visual feedback.
- **Immersive Audio**: Custom synthesized alarm sounds using the Web Audio API (no external assets required).
- **Responsive Design**: Works beautifully on both desktop and mobile browsers.

## 🚀 How to Use

1. **Clone or Download** this repository.
2. Open `index.html` in any modern web browser (Chrome, Safari, Firefox, or Edge).
3. Click **시작 (Start)** to begin your focus session.

## 🧪 Testing Shortcut

Want to see the end-of-session celebration without waiting 25 minutes?
- **Double-click** anywhere on the large timer text ("25:00") to immediately jump to the last 5 seconds of the session.

## 🛠️ Technical Stack

- **HTML5 / CSS3**: Vanilla CSS for all animations and styling (no frameworks).
- **ES6+ JavaScript**: Logic for timer, state management, and SVG manipulation.
- **SVG**: For the high-precision character and progress visualization.
- **Web Audio API**: For high-fidelity sound synthesis.

# 🍎 애플 스타일 뽀모도로 졸라맨 타이머 (README.md)

애플의 미니멀한 디자인 철학을 담은 우아한 단일 파일 뽀모도로 타이머입니다. 생동감 넘치는 졸라맨 캐릭터가 여러분의 집중 시간과 휴식 시간을 시각적으로 즐겁게 안내해 드립니다.

## ✨ 주요 기능

- **애플 미니멀리즘**: 글래스모피즘, 부드러운 그라데이션, 세련된 타이포그래피가 적용된 프리미엄 UI.
- **역동적인 졸라맨 애니메이션**:
    - **0-15초 (휴식)**: 댄스, 기지개, 발 구르기 등 위트 있는 랜덤 동작을 보여줍니다.
    - **15-50s (공전)**: 현재 분(Minute)을 나타내는 점 주위를 궤도를 그리며 걸어가며 시간의 흐름을 표현합니다. (중간 과정의 생동감)
    - **50-60s (전력 질주)**: 남은 궤도를 전력 질주하며 다음 점으로 넘어갈 준비를 합니다.
    - **60s (도약!)**: 1분이 되는 순간, 탄력 있는 '보잉' 효과와 함께 다음 점으로 폴짝 뛰어 넘어갑니다.
- **원형 진행 표시**: 집중 시간(25개)과 휴식 시간(5개)을 동그라미 배열로 직관적으로 표시합니다.
- **실감나는 오디오**: 별도의 파일 없이 Web Audio API로 직접 합성한 경쾌한 알람 사운드.
- **반응형 디자인**: 데스크탑과 모바일 어디서나 최적의 레이아웃을 제공합니다.

## 🚀 사용법

1. `index.html` 파일을 웹 브라우저(Chrome, Safari 등)로 엽니다.
2. **시작** 버튼을 눌러 타이머를 작동시킵니다.

## 🧪 테스트 팁

25분을 다 기다리지 않고 애니메이션을 확인하고 싶을 때:
- 화면 중앙의 커다란 타이머 숫자("25:00")를 **더블 클릭**하면 즉시 종료 5초 전으로 이동하여 세리머니와 알람을 테스트할 수 있습니다.

## 🛠️ 기술 스택

- **HTML5 / CSS3**: 프레임워크 없는 순수 CSS 애니메이션 및 스타일링.
- **ES6+ JavaScript**: 타이머 로직, 상태 관리 및 SVG 조작.
- **SVG**: 고정밀 캐릭터 및 진행률 시각화.
- **Web Audio API**: 고품질 사운드 합성.
